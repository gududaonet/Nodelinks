# Nodelinks
免费链接太长背不下来？第三方服务担心隐私？用自己的站点聚合害怕喝茶？
---
**Nodelinks** 是一个轻量级工具，专为 **Cloudflare Workers** 设计，用于聚合多个 V2Ray 订阅链接。  
本软件支持**裸 Vmess 格式**和**加密格式**的订阅链接，提供**隐私保护**，并支持**密码访问控制**。
---
## ✨ 功能特性
- 🚀 完全运行于 Cloudflare Workers（无服务器，零维护）
- 🔐 通过 `key` 参数进行密码保护访问
- 🔀 支持多模式聚合订阅链接
- 🌐 拉取并合并多个在线订阅链接
- 📄 可返回纯文本或 HTML 页面
- 🧩 轻量、易于定制和扩展
---
## 🧩 支持模式
### 选择节点组（固定链接聚合）
- 预定义订阅链接会自动合并，可用于选择你的节点组
- 访问示例：
```
/?mode=1&key=YOUR_PASSWORD
```
---
## 🔐 鉴权说明
- 所有请求必须提供正确的 `key` 参数
- 未授权请求将返回 `ACCESS DENIED`
- 在 403 页面上，不会暴露这是一个聚合脚本
---
## 📦 部署说明
### 1️⃣ 创建 Cloudflare Workers 项目
### 2️⃣ 提交 `index.js`
将 Workers 脚本代码复制到项目的 `index.js` 文件中
### 3️⃣ 修改代码
添加你的订阅链接，并填写密码
### 4️⃣ 发布
---
## ⚙️ 配置提示
- 确保你的 URL 正确，并且 Cloudflare 服务器不会拒绝访问
- 推荐的在线订阅聚合（可以使用Workers调用）[https://sub.dler.io/](https://sub.dler.io/)
---
## ⚠️ 注意事项
- 本项目 **不依赖 Node.js API**，仅使用标准 **Workers Web API**
- 不适合高频或大规模抓取
- 确保所有被抓取的 URL 可以访问
